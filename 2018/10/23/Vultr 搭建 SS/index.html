以下教程基于 vultr VPS，系统是 ubuntu14.04

一、搭建 SS

要使用 SS 需要购买国外的 VPS，这里推荐 vultr 和 linode。因为锐速 serverspeeder 目前只支持 centos 和 ubuntu 的一些内核版本，所以这里建议系统选择 ubuntu14.04。购买完成后进入 Ubuntu 系统，一条一条地输入以下命令

安装过程中会要求你输入SS密码、端口号(默认8989)和选择加密算法，选择 aes-256-cfb (输入7)就可以了，如图，这样手机上也能使用SS。

等待片刻会显示 SS 的一些信息(如图红色区域)，这些信息我们需要输入到 shadowsocks 中来连接 VPS，所以最好保存下来。

四、配置 shadowsocks

下载 github 上的 shadowsocks-windows，打开后输入服务器地址、服务器端口和密码，加密算法选择 aes-256-cfb，确定后托盘纸飞机图标右键启用系统代理就可以翻墙啦！
安卓可以下载 shadowsocks-android，配置一下就可以了！
iOS 商店搜索 Wingy ，配置一下就可以了！

后记

以上安装的是Python版的SS，默认创建的是单用户配置。若要创建多用户配置可参考 Shadowsocks-go一键安装脚本 或者 Shadowsocks Python版一键安装脚本
